AutoCAD Task 1 – Base Plate Drawing

 Description

This project contains a 2D mechanical drawing of a rectangular base plate created using AutoCAD.

 📐 Specifications

* Length: 200 mm
* Width: 120 mm
* Thickness: 10 mm
* Hole Diameter: 12 mm
* 4 Holes at corners

 🛠️ Tools Used

* AutoCAD
* Mechanical Drafting Standards

📂 Files Included

* DWG File (AutoCAD drawing)
* PDF Export
* Layer Manager Screenshot
* Dimensioned Drawing Screenshot

📷 Output

Includes Top View and Front View with proper dimensions and layers.
